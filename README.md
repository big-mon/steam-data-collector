# Steam Data Collector

Steamの[StorefrontAPI](https://wiki.teamfortress.com/wiki/User:RJackson/StorefrontAPI)を利用して、ストア情報を収集します。

前提としてMySQLに接続可能であり、DBが作成済みであることが必要です。
前提となるDBは[big-mon/steam-omakuni](https://github.com/big-mon/steam-omakuni)と連動させることで自動的に実装が可能です。

## Getting Started

Visual Studioなどを使用して、`steam-data-collector.sln`をビルドしてください。
